# Blog Platform

A full-stack **Blog Platform** built with **Node.js, Express, MongoDB, and Mongoose**.  
This platform allows users to **register/login, create blog posts, comment on posts, like posts/comments**, and filter posts by **tags, categories, authors**, with **pagination and sorting support**.

## Features

- User registration and login with **hashed passwords** using **bcrypt**  
- Create, read, update, and delete **blog posts**  
- Add, read, and delete **comments** on posts  
- Like/upvote **posts and comments**  
- Filter posts by **tags, category, and author**  
- **Pagination** and **sorting** for posts and comments  
- Fully seeded database for testing/demo purposes  

---

## Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose  
- **Authentication:** bcrypt (password hashing)  
- **Environment Variables:** dotenv  
- **Frontend:** Vanilla JavaScript, HTML, CSS  
- **Version Control:** Git, GitHub
